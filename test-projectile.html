Hunter Projectile Test

1. Open browser console (F12)

2. Start server: npm start

3. Open game in browser

4. Choose Hunter class

5. Press Left Mouse Button to shoot

6. Check console for:

  • "Primary attack (bow shot) started"
  • "Executing projectile attack for hunter primary"
  • "Creating projectile at..."
  • "Sending createProjectile to server:"

7. Check server console for:

  • "Received createProjectile request from..."
  • "Projectile X created by..."
